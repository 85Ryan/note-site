## if 语句

`if`语句语法：

```js
if (condition) {
  // Code to run when the condition is true.
}
```

{% note warn %}
当只有一句条件语句时，可以省略花括号（`{...}`），但对初学者来说，应该始终使用花括号。
{% endnote %}
